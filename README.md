## Dependencias y Compilación

Paso 2: Compile primero para implementación local web app .

```
cd client 
npm install (instalación de dependencias)
npm run start
(port 3000)
```

Paso 3: Si va a compilar el backend.

```
cd api 
npm install (instalación de dependencias)
npx nodemon server.js
(port 5000)
```
